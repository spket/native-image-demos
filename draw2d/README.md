build source
```
./build.sh
```
run and generate configuration for native image
```
./run.sh -trace
```
build native image
```
./build.sh -native
```
launch 
```
./draw2dapp
```
Blog: [Graalvm Native Image Demo - JFace and Draw2D](https://www.spket.com/blog/graalvm-native-image-demo-jface-and-draw2d/)
